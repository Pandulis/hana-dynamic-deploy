## Excluding this file.

This README.md file can exist in the db/src folder due to an additon in the .hdiconfig file.  See the bottom of the .hdiconfig file.

```
        ...
        "txt" : {
            "plugin_name" : "com.sap.hana.di.copyonly"
        },
        "md" : {
            "plugin_name" : "com.sap.hana.di.copyonly"
        }
```
